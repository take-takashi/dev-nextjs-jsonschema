# dev-nextjs-jsonschema
Next.jsでJsonSchemaを試したい

## install command
```
npx create-next-app nextapp --ts --use-npm
cd nextapp/
npm install @rjsf/core @rjsf/utils --save
npm install @rjsf/validator-ajv6 --save
```
